# Opik

***

## Setup

1. At the top right corner of your Chatflow or Agentflow, click **Settings** > **Configuration**

<figure><img src="https://823733684-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F00tYLwhz5RyR7fJEhrWy%2Fuploads%2Fgit-blob-c4693d4dc7a34d3203ee60dc442eaeacb6798b22%2Fanalytic-1.webp?alt=media" alt="Screenshot of user clicking in the configuration menu" width="375"><figcaption></figcaption></figure>

2. Then go to the Analyse Chatflow section

<figure><img src="https://823733684-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F00tYLwhz5RyR7fJEhrWy%2Fuploads%2Fgit-blob-93e50237b8df24cb113a69c3b9e04487527ccadd%2Fanalytic-2.png?alt=media" alt="Screenshot of the Analyse Chatflow section with the different Analytics providers"><figcaption></figcaption></figure>

3. You will see a list of providers, along with their configuration fields. Click on Opik.

<figure><img src="https://823733684-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F00tYLwhz5RyR7fJEhrWy%2Fuploads%2Fgit-blob-906b04a32db87240873de6060514fdb4558e5068%2Fopik-1.png?alt=media" alt="Screenshot of an analytics provider with credentials fields expanded"><figcaption></figcaption></figure>

4. Create credentials for Opik. Refer to the [official guide](https://www.comet.com/docs/opik/tracing/sdk_configuration) on how to get the Opik API key.

<figure><img src="https://823733684-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F00tYLwhz5RyR7fJEhrWy%2Fuploads%2Fgit-blob-a561e4c7ea1a4abeb94e77fd8b7364b8cbf27d32%2Fopik-2.png?alt=media" alt="Screenshot of analytics providers enabled"><figcaption></figcaption></figure>

5. Fill in other configuration details, then turn the provider **ON**

<figure><img src="https://823733684-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F00tYLwhz5RyR7fJEhrWy%2Fuploads%2Fgit-blob-88ae642219fbd6101eedd7e5aac45f77fc9f1a2e%2Fopik-3.png?alt=media" alt="Screenshot of analytics providers enabled"><figcaption></figcaption></figure>

Now you can analyze your chatflows and agentflows using Opik UI:

<figure><img src="https://823733684-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F00tYLwhz5RyR7fJEhrWy%2Fuploads%2Fgit-blob-69d816d5dfb8dc47280b6da0069773985acdb8fe%2Fopik-4.png?alt=media" alt="Screenshot of Opik UI"><figcaption></figcaption></figure>
